# Graph Traversal

> [!summary]
> Graph Traversal 主要研究如何系统地访问图中的点和边。  
> DFS / BFS 是基础工具，很多进阶主题例如拓扑排序、判环、二分图、割点桥、SCC 和 2-SAT 都建立在它们之上。

1. Start with [[Cycle Detection in Directed Graphs]] and [[Topological Sort (DFS Kahn)]].
2. Then study [[Bipartite Graph Check]] for undirected graph coloring.
